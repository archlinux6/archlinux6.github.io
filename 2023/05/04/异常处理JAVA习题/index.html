简单异常处理

【问题描述】

输入2个整数，输出2个整数相除的结果。

要求：

（1）若输入的不是整数（如输入字符串或实数），程序捕获异常（InputMismatchException），并输出异常信息“format error”

（2）若输入的第2个数据为0，程序捕获异常（ArithmeticException），并输出异常信息“0 error”

无论是否出现异常，程序结束均要输出“over”

【输入形式】

两个数据

【输出形式】

运算结果

【样例输入1】

4 5

【样例输出1】

0

over

【样例输入2】

5 3.5

【样例输出2】

format error

over

【样例输入3】

5 0

【样例输出2】

0 error

over

【样例说明】
【评分标准】

必须采用异常处理机制解决。

构造方法：初始化name和age；

公有方法：

（1）setAge方法：修改age值；

（2）getAge方法：返回age值；

（3）getName方法：返回name值。

要求抛出异常：

（1）创建对象时，检查name和age。若name为null或空或空格，则抛出异常，异常信息为“name null”；若age<6或age>100，则抛出异常，异常信息为“age error”;

(2) 更新age时，若age<6或age>100，则抛出异常，异常信息为“age error”;

Student类，只抛出异常，不处理。
