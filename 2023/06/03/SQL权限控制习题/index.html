将给定的 emp_num 值加1，如果 emp_num 是空值，则将其赋值为0再加1。
COALESCE() 函数用于处理NULL值，它返回参数列表中的第一个非空值。
在这里，如果 emp_num 是 NULL 值，那么 COALESCE 将会返回 0，然后将 0 加上 1 得到 1。
如果 emp_num 不是 NULL 值，则将给定的 emp_num 值加上1。
